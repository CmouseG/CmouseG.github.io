Hexo 个人博客环境搭建

发表于 2017-02-22 | 分类于 IT ， Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

基础篇

前提

安装Hexo是比较简单的，但安装之前必须安装好Node.js，Git

安装

就是这么简单

使用

一个博客项目就建成了，下目录如下

├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

运行服务

在浏览器中输入localhost:4000就可以博客展示效果了

新建一篇博文

设置主题

推荐使用NexT主题
安装主题

启用主题
修改站点配置文件

theme: next

添加标签

确认站点配置文件tag_dir: tags
确认主题配置文件tags: /tags
编辑站点的source/tags/index.md，添加

title: 标签标题
date: 2017-02-22 21:42:40
type: tags
comments: false # 如果开启评论模块

添加分类

确认站点配置文件category_dir: categories
确认主题配置文件categories: /categories
编辑站点的source/categories/index.md，添加

title: 分类标题
date: 2017-02-22 21:43:46
type: categories
comments: false # 如果开启评论模块

部署

首先要有GitHub或Coding帐号，创建GitHub Pages或Coding Pages
安装hexo-deployer-git

修改站点配置文件(_config.yml)

deploy:

type: git
github: git@github.com:{username}/{username}.github.io.git,{branch[master]}
coding: git@git.coding.net:{username}/{project}.git,{branch[master]}
