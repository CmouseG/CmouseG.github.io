GitHub、Coding个人博客映射域名

发表于 2017-02-25 | 分类于 IT ， 网络 |

Hexo部署个人博客到GitHub或Coding后，就可以直接在浏览器在输入{account}.github.io或{account}.coding.me/{project}直接访问了，Coding还不能正常访问，因为一些静态资源会访问不到，导致页面显示不正常，如果想用一个自定义域名指向你的博客，并且方便百度收录，你可以映射你的域名。

域名注册和解析

添加映射

GitHub:
GitHub添加映射比较简单，直接在根目录下新建一个CNAME文件，注意不要有后缀，内容为你刚注册并解析后的域名

Coding:
到项目下，代码—>Pages服务—>自定义域名，输入你刚注册并解析后的域名，然后绑定，等一会会显示绑定状态，记得要先要注册和解析域名，否则会申请失败，最后你就有3个月的试用了。
